## 📦 Installation

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Local Development

1. **Clone the repository**
```bash
git clone https://github.com/ahaobrianai/ahaobrian-ai-platform.git
cd ahaobrian-ai-platform
```

2. **Install dependencies**
```bash
npm install
```

3. **Start the server**
```bash
npm start
```

4. **Open in browser**
```
http://localhost:3000
```

### Development Mode (with auto-reload)
```bash
npm run dev
```

## 🚀 Deployment

### Deploy to Vercel (Recommended)

1. **Install Vercel CLI**
```bash
npm i -g vercel
```

2. **Deploy**
```bash
vercel
```

3. **Production deployment**
```bash
vercel --prod
```

Your platform will be live at: `https://your-project.vercel.app`

### Deploy via Vercel Dashboard

1. Go to [vercel.com](https://vercel.com)
2. Import your GitHub repository
3. Click "Deploy"
4. Done! 🎉

## 📱 Usage

### AI Chat
1. Click on "AI Chat Assistant"
2. Type your message
3. Get instant AI-powered responses

2. Describe your desired image
3. Wait for AI to create it

2. Enter movie title
